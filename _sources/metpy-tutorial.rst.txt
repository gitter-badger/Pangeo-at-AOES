About MetPy
#############

Python tools for weather and climate data

Website: https://unidata.github.io/MetPy/


What is MetPy?
***************

A collection of tools in Python for reading, visualizing, and performing calculations with weather data.
It is developed and is maintained by the fine folks at Unidata, a part of UCAR that has been providing data services and tools for our profession since 1983.


Installation
**************

If you followed the `Getting Started` section, then you already have MetPy installed. 

Otherwise, you can install with conda

.. code-block:: bash

  conda install -c conda-forge metpy

or pip

.. code-block:: bash

  pip install metpy

What is it good for?
********************

MetPy is particularly good at handling , checking, and converting units for meterological data.
